## 附一、中试调研数据一页纸（打印给老师的时效性数据）

> **用途：** 答辩时老师追问「数据哪来的、为什么现在做」时递阅；与 PPT 第 4 页、口述稿 **数据一致**。  
> **原则：** 一页 A4、**有年份、有出处**；数字加粗；背景数据与系统测试数据 **分开注明**。  
> **答辩前：** 重新打开 §附一 末「来源索引」链接 **核对** 是否仍有更新版本。

### 附1.1 页面设置（Word / WPS）

| 项 | 建议值 |
|----|--------|
| 纸张 | **A4 纵向** |
| 页边距 | 上 **1.8 cm** · 下 **1.8 cm** · 左 **2.0 cm** · 右 **2.0 cm** |
| 正文字体 | **宋体 9 pt**（英文/数字可用 Arial 9 pt） |
| 一级标题 | **黑体 12 pt 居中** |
| 二级小标题 | **黑体 10 pt**（如「国家层面」「广州市」） |
| 行距 | **固定值 14 pt** 或 **单倍行距**（勿用 1.5 倍，易超一页） |
| 段前段后 | **0 行** |
| 打印份数 | **2—3 份**（评委多位时够用；可夹在开题报告里） |
| 颜色 | **黑白打印**即可；关键数字可用 **加粗**，不必彩打 |

### 附1.2 版式结构（一页五区）

老师 **3—5 秒** 能扫完：上面是 **分层数据**，中间是 **口述用核心数字**，下面是 **选题逻辑** 和 **来源索引**。

```
┌──────────────────────────────────────────────────────────────┐
│  ① 页眉（1 行标题 + 1 行副信息）                               │
├──────────────────────────────────────────────────────────────┤
│  ② 三栏正文（国家 | 省/市 | 甲方）  约各占 1/3 宽，无竖线亦可   │
│     每栏 3—4 条 • 条目，每条「数据 +（年份）+ 来源序号」        │
├──────────────────────────────────────────────────────────────┤
│  ③ 核心数字条（1 行，答辩可口述）                              │
├──────────────────────────────────────────────────────────────┤
│  ④ 选题逻辑（1—2 行，与 PPT 第 4→5 页衔接）                    │
├──────────────────────────────────────────────────────────────┤
│  ⑤ 页脚：来源 [1]—[9] 缩写 + 核对说明 + 模拟数据声明           │
└──────────────────────────────────────────────────────────────┘
```

**Word 操作要点：**

1. 标题下插入 **表格 1 行 3 列**（无边框或仅浅灰底纹），填「国家层面 / 广东省·广州市 / 甲方·中心」。  
2. 三栏内一律 **• 条目**，**数字加粗**；来源写 **`[1]`** 这类序号，详细出处放页脚。  
3. 「核心数字条」可用 **浅灰底纹单行表格** 或加粗独立段落。  
4. 页脚来源用 **8 pt 宋体**，两列排版（左 [1]—[5]，右 [6]—[9]）更省行。  
5. 排完后 **打印预览**：必须 **整页在一页内**；超出则删次要修饰词或将页脚改为 8 pt。
